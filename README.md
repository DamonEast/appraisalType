## appraisaType

Js类型检测

## 何时使用

项目中检验数据类型。

## 代码展示

import AppraisalType from 'appraisal-type'

AppraisalType(123456, 'number)

## API Lable

| 参数              | 说明                                       | 类型       | 默认值      |
| --------------- | ---------------------------------------- | -------- | -------- |
| targetData          | 需要检测的数据                              | *   |        |
| type       | 传入的类型                           | string   | -        |
